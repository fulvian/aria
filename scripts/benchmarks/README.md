# Benchmark Scripts

## MCP Startup Latency Benchmark

`mcp_startup_latency.py` — Misura cold/warm startup + tools/list per ogni MCP server.

```bash
.venv/bin/python scripts/benchmarks/mcp_startup_latency.py          # markdown output
.venv/bin/python scripts/benchmarks/mcp_startup_latency.py --json   # JSON output
.venv/bin/python scripts/benchmarks/mcp_startup_latency.py --search-only  # solo search domain
```
